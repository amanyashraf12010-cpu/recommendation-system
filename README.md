# 🍔 Recommendation System

A hybrid recommendation system that combines:

- Collaborative Filtering (SVD)
- Content-Based Filtering (TF-IDF)
- Context-Aware Scoring (price, expiry, discount)

## 🚀 Features
- Personalized recommendations
- Cold-start handling
- FastAPI deployment
- Ready for mobile/web integration

## 📊 Tech Stack
- Python
- Scikit-learn
- Pandas
- FastAPI
